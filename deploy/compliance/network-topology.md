## 拓扑结构

```
用户浏览器（HTTPS）
    │
    ▼
DNS 解析（域名 → 服务器 IP）
    │
    ▼
Nginx 反向代理（80/443 端口）
    ├── /api/* → Express 后端（127.0.0.1:3001）
    ├── /ws → WebSocket 后端（127.0.0.1:3001）
    └── /* → 前端静态文件（client/dist/）
            │
            ▼
        SQLite 数据库（本地文件存储）
```

## 服务端口清单

| 端口 | 协议 | 用途 | 对外开放 |
|------|------|------|---------|
| 22 | SSH | 服务器管理 | 否（仅管理员 IP） |
| 80 | HTTP | 网站入口 → 跳转 HTTPS | 是 |
| 443 | HTTPS | 网站加密访问 | 是 |
| 3001 | HTTP | Express 后端（仅本地） | 否（仅 127.0.0.1） |

## 数据存储
- SQLite 数据库文件存储在服务器本地 `/home/ubuntu/class-pet-garden/server/data/`
- 用户密码使用 bcrypt 加密存储
- 不对外提供数据库直连端口
