# 班级宠物园 🐱

> 游戏化班级管理工具 —— 把积分变成萌宠养成，让班级管理更有趣！

---

## 一、产品定位

### 给谁用
中小学班主任、科任老师、少先队辅导员

### 解决什么问题
传统的班级积分评价制度枯燥、学生积极性不高。班级宠物园将**积分评价**与**电子宠物养成**结合，学生获取的每一分都能用于喂养自己的宠物，宠物会随着积分积累而成长进化，从而让学生从"被动接受评价"变为"主动争取积分"。

### 核心玩法
```
老师给学生加/减分 → 学生积分变化 → 学生用积分喂养宠物 → 宠物获得经验升级 → 解锁新形态
```

### 8 级成长

| 等级 | 名称 | 所需经验 | 形态描述 |
|------|------|---------|---------|
| Lv.1 | 初生 | 0 - 99 | 破壳而出，娇小可爱 |

| Lv.3 | 少年 | 300 - 599 | 特征逐渐鲜明 |
| Lv.4 | 青年 | 600 - 999 | 颜色鲜亮，充满活力 |
| Lv.5 | 成熟 | 1,000 - 1,499 | 完全成年形态 |
| Lv.6 | 精英 | 1,500 - 2,499 | 散发微光特效 |
| Lv.7 | 王者 | 2,500 - 3,999 | 获得王冠或专属装饰 |
| Lv.8 | 传说 | 4,000+ | 满级特效 + 专属徽章 |

### 喂养规则
- 学生积分 = 喂养货币，1 积分 = 1 经验
- 每次喂养可选 5 / 10 / 20 / 50 积分
- 喂养后立即扣积分、加经验、检测是否升级
- 每名学生最多领养 5 只宠物
- 不能重复领养同一种宠物

## 五、技术架构

| 层 | 技术 | 说明 |
|---|---|---|
| 前端 | React 18 + Vite + TypeScript + Tailwind CSS | 浏览器访问，无需安装 |
| 后端 | Node.js + Express + TypeScript | 处理数据请求和登录认证 |
| 数据库 | SQLite（sql.js） | 单文件数据库，免安装配置 |
| 认证 | bcrypt + JWT | 教师账号密码登录 |

### 项目结构
```
class-pet-garden/
├── README.md           ← 产品需求文档（你正在看）
├── shared/             ← 前后端共用的数据和规则（宠物种类、等级表）
├── server/             ← 后端服务（API 接口、数据库）
├── client/             ← 前端界面（网页页面）
├── package.json        ← 项目配置
└── .claude/plans/      ← Claude Code 开发计划
```
