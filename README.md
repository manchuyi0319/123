# 我的老师我的班 🐱

> 游戏化班级管理工具 —— 把积分变成萌宠养成，让班级管理更有趣！

---

## 一、产品定位

### 给谁用
中小学班主任、科任老师、少先队辅导员，以及学生家长

### 解决什么问题
传统的班级积分评价制度枯燥、学生积极性不高。「我的老师我的班」将**积分评价**与**电子宠物养成**结合，学生获取的每一分都能用于喂养自己的宠物，宠物会随着积分积累而成长进化，从而让学生从"被动接受评价"变为"主动争取积分"。

### 核心玩法
```
老师给学生加/减分 → 学生积分变化 → 学生用积分喂养宠物 → 宠物获得经验升级 → 解锁新形态
家长通过邀请码关联孩子 → 查看积分/宠物/排名 → 参与竞赛闯关
```

### 喂养规则
- 每次喂养消耗 5 积分，随机获得 10-30 经验
- 每名学生最多领养 5 只宠物
- 不能重复领养同一种宠物

## 四、竞赛闯关规则

- 题库包含各年级学科知识、益智类、脑筋急转弯类，共 999 道题目
- 每题答对升 1 关，答错停留在当前关卡
- 每过 10 关奖励 50 积分（可用于喂养宠物）
- 每天最多 3 次答错机会，次日上午 **9:00** 重置
- 年级与注册班级关联，题库自动匹配对应难度

## 六、技术架构

| 层 | 技术 | 说明 |
|---|---|---|
| 前端 | React 18 + Vite + TypeScript + Tailwind CSS | SPA + PWA 可安装 |
| 后端 | Node.js + Express + TypeScript | RESTful API |
| 数据库 | SQLite（sql.js WASM） | 单文件数据库，免安装 |
| 认证 | bcryptjs + JWT | Bearer Token 认证 |
| 验证 | Zod | 请求参数校验 |
| 数据获取 | SWR | 前端数据 fetching + 缓存 |
| 包管理 | npm workspaces | monorepo（shared/server/client） |

### 数据库表（15 张）
- `teachers` — 教师/家长账号（含角色、金币余额）
- `classes` — 班级（含邀请码、软删除）
- `students` — 学生（含积分、竞赛关卡进度）
- `pets` — 宠物图鉴（23 种种子数据）
- `student_pets` — 领养记录
- `point_records` — 积分流水
- `evaluation_rules` — 评价规则
- `parent_students` — 家长-学生关联
- `join_requests` — 家长加入申请
- `payment_orders` — 支付订单
- `recharge_codes` — 充值码
- `coin_records` — 金币流水
- `quiz_questions` — 竞赛题库
- `quiz_records` — 答题记录
- `quiz_daily_stats` — 每日答题统计
- `announcements` — 系统公告
- `bulletin_posts` — 论坛帖子
- `semester_rewards` — 学期奖励配置
- `class_promotions` — 班级升学记录
